# vue-devtools-plus

Vue Devtools, but better.

`vue-devtools-plus` is a customized fork of Vue Devtools v5.3.4 focused on improving the Vue development experience.

## Development

```bash
yarn install
yarn run dev
```

## Build

```bash
yarn run build
```

## Chrome Extension

Build the project, then load `packages/shell-chrome/` as an unpacked extension in Chrome.
